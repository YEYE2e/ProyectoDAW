# Aura | Premium Tees
**Proyecto 1 DAW 2026**

Landing page moderna y minimalista para una tienda de camisetas de alta calidad. El proyecto es un sitio estático de alto rendimiento centralizado en un único archivo HTML.

## Características
- **Arquitectura Simple**: Sitio contenido íntegramente en `index.html`.
- **Diseño Premium**: Estética limpia con tipografía Outfit y animaciones CSS3.
- **Carrito Funcional**: Lógica de compras con Vanilla JavaScript y notificaciones dinámicas.
- **Responsivo**: Optimizado para dispositivos móviles, tablets y escritorio.

## Tecnologías
- HTML5 Semántico
- CSS3 (Variables, Flexbox y Grid)
- Vanilla JavaScript
- FontAwesome 6.0 (Iconos)
- Google Fonts (Fuentes)

---

## Despliegue en GitHub Pages

El proyecto se despliega automáticamente mediante GitHub Pages siguiendo estos pasos:

1. Asegurarse de que todos los archivos estén en la rama principal del repositorio.
2. En GitHub, ir a **Settings** > **Pages**.
3. En "Source", seleccionar la rama `main` y la carpeta `/ (root)`.
4. El sitio será accesible en la URL proporcionada por GitHub.
